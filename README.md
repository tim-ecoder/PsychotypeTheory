## Theory

Eight psychotypes arranged in a circle. Step distance between them determines the interaction type. No arbitrary assignments.

### Psychotypes

| ID | Name | Trait |
|---|---|---|
| PAR | Paranojal | Goal-driven, systemic, willful |
| EPI | Epileptoid | Orderly, accumulating, regulatory |
| IST | Hysteroid | Demonstrative, expressive, recognition-seeking |
| EMO | Emotiv | Sensitive, empathic, conflict-avoidant |
| GIP | Hyperthymic | Active, optimistic, high-energy |
| SHZ | Schizoid | Introverted, conceptual, nonlinear |
| GOT | Hypothymic | Pessimistic, cautious, low-affect |
| TRV | Anxious | Vigilant, risk-averse, threat-oriented |

### 4 Interaction Types

| Step | Type | Direction | Pairs |
|------|------|-----------|-------|
| 1 | Generation | one-way | 8 (full cycle) |
| 2 | Polarity | mutual | 8 (across 2 squares) |
| 3 | Suppression | one-way | 8 |
| 4 | Cooperation | mutual | 4 (diametrically opposite) |

**32 pairs total, 100% theoretical consistency.**

### Polarity Squares (step 2)

- Extraverted: PAR - IST - GIP - GOT
- Introverted: EPI - EMO - SHZ - TRV

## Visualization

Single self-contained HTML file. No dependencies, no build step.

**Features:**

- Hover a node: psychotype description + all connections in context
- Hover an arrow: unique description for that specific pair interaction
- 4 semantic presets (Classic / Love / Energy / Therapy): same geometry, different lens
- Toggle any interaction layer on/off independently
- Click a node to pin highlighting

**Run locally:**

    git clone https://github.com/tim-ecoder/PsychotypeTheory.git
    open psychotypes_theory.html
